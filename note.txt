# PabloTours Website Prompt Engineering Documentation

Prompt 1

Create a modern, premium travel agency website called **PabloTours** using HTML and Tailwind CSS. The website should contain 10 interconnected pages and present a luxury travel experience. Use a clean, elegant interface with a soft blue and beige color palette. Ensure the design is responsive and visually appealing on desktop, tablet, and mobile devices.
